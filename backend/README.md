# NovaPrep Backend

See the [main README](../README.md) for complete setup and run instructions.

## Quick Reference

- **Start**: `npm run dev`
- **Port**: `http://localhost:4000`
- **Endpoints**: `/api/generate-question`, `/api/evaluate-answer`, `/api/final-summary`, `/api/parse-resume`
- **Voice**: `/api/voice/transcribe`, `/api/voice/speak`
